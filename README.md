# Bedrock Chat

[![Build](https://github.com/Axel-Jalonen/bedrock-chat/actions/workflows/rust.yml/badge.svg)](https://github.com/Axel-Jalonen/bedrock-chat/actions/workflows/rust.yml)
[![Release](https://img.shields.io/github/v/release/Axel-Jalonen/bedrock-chat?label=download)](https://github.com/Axel-Jalonen/bedrock-chat/releases/latest)

A native desktop chat client for AWS Bedrock. Rust, egui, ~1.6k lines.

> [**Download latest release**](https://github.com/Axel-Jalonen/bedrock-chat/releases/latest)
>
> The binary is **unsigned**. macOS will block it. To run it:
> 1. Download the `.tar.gz` for your architecture (arm64 for Apple Silicon, x86_64 for Intel)
> 2. Extract: `tar xzf bedrock-chat-macos-*.tar.gz`
> 3. Remove the quarantine flag: `xattr -d com.apple.quarantine bedrock-chat-macos-*`
> 4. Run: `./bedrock-chat-macos-*`
>
> Or just build from source (see below).

## Disclaimer

This entire codebase was generated by an LLM in a single conversation. It has been lightly tested but not audited. The binary is unsigned and unnotarized. It stores your API key in a local SQLite database at `~/.config/bedrock-chat/history.db` in plaintext. Use at your own risk. The author disclaims all liability for anything this software does, fails to do, or causes to happen.

## What it does

- 64 models across 16 providers (Anthropic, Meta, DeepSeek, Mistral, Qwen, Amazon Nova, Google, NVIDIA, etc.)
- Searchable model picker, grouped by provider
- Startup modal: paste your Bedrock API key and go, or skip for existing AWS config
- Streaming responses via ConverseStream with token usage tracking
- Markdown + code blocks via egui_commonmark
- Chat search (Cmd+K) across all conversations
- Ephemeral mode: chat without writing to disk
- Compact context: summarize conversation history to reduce token usage
- Region locked after first message (prevents mid-conversation breakage)
- System theme support (follows macOS dark/light mode)
- Particle effects with connections, glow, parallax, and mouse interaction
- Persists to SQLite at `~/.config/bedrock-chat/history.db`

## Build from source

Needs Rust 1.92+ and macOS. Get a Bedrock API key from the [AWS console](https://console.aws.amazon.com/bedrock/) and paste it on startup.

```bash
cargo run --release
```

## Releasing

Push a tag to create a release:

```bash
git tag v0.1.0
git push origin v0.1.0
```

This triggers the release workflow which builds macOS binaries (arm64 + x86_64) and creates a GitHub Release with the artifacts attached.

## Structure

```
src/main.rs       ~35 lines   entry point
src/app.rs       ~900 lines   UI, search, compact, ephemeral, particles
src/bedrock.rs   ~140 lines   streaming client + token usage
src/db.rs        ~220 lines   sqlite + search
src/message.rs   ~200 lines   types + all 64 model definitions
```
