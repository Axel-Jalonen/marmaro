# Bedrock Chat

> **Alpha Release** - This is an early alpha. Expect bugs, missing features, and breaking changes.

[![Build](https://github.com/Axel-Jalonen/bedrock-chat/actions/workflows/rust.yml/badge.svg)](https://github.com/Axel-Jalonen/bedrock-chat/actions/workflows/rust.yml)
[![Release](https://img.shields.io/github/v/release/Axel-Jalonen/bedrock-chat?label=download)](https://github.com/Axel-Jalonen/bedrock-chat/releases/latest)

Native macOS chat client for AWS Bedrock. Built with Rust + egui.

[**Download latest release**](https://github.com/Axel-Jalonen/bedrock-chat/releases/latest) (Apple Silicon). The binary is unsigned - macOS will complain. Go to System Settings > Privacy & Security > Open Anyway.

**Warning:** This was generated by an LLM. Not audited, not signed, not notarized. Stores your API key in plaintext in `~/.config/bedrock-chat/history.db`. Use at your own risk.

## Features

- **64 models** from 16 providers (Claude, Llama, DeepSeek, Mistral, Qwen, Nova, etc.)
- Searchable model picker with keyboard navigation
- Streaming responses with token usage display
- Markdown rendering with code blocks
- Chat history with full-text search (Cmd+K)
- System prompts per conversation
- Ephemeral mode (nothing saved to disk)
- Context compaction (summarize history to save tokens)
- Double-tap Esc to interrupt generation
- Smart scroll - read while streaming without being yanked back down
- Follows system dark/light theme

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| Enter | Send message |
| Shift+Enter | New line |
| Cmd+K | Search chats |
| Esc | Close modal |
| Esc Esc | Interrupt generation |
| Arrow keys | Navigate lists |

## Build from source

```bash
cargo run --release
```

Requires Rust 1.75+ and macOS.
