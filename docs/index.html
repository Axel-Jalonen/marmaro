Pure Rust

Built with egui. No Electron, no WebView, no JavaScript.

LaTeX rendering uses a patched fork of ReX, a pure Rust TeX layout engine. The original hadn't been updated for modern Rust, so we fixed the build and added support for commands LLMs commonly output: \text, \operatorname, \boxed, Cyrillic characters, etc.

Markdown parsing uses pulldown-cmark with a custom renderer that interleaves styled text and math on the same line.

Credentials are stored in the macOS Keychain.
